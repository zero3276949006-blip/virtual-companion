## 快速开始

### 第一次使用

当你第一次和 AI 聊到这个技能时，会自动启动引导流程：

1. **讲述你的故事** — 陪伴者会用温暖的开场白和你打招呼
2. **介绍自己** — 告诉陪伴者你的名字
3. **选择关系** — 告诉陪伴者你们是什么关系
4. **塑造陪伴者** — 告诉陪伴者它应该是什么性格
5. **设定偏好** — 是否开启早安晚安等互动仪式

### 常用指令

```
/companion profile    查看当前陪伴者档案
/companion list      列出所有陪伴者
/companion switch    切换到另一个陪伴者
/companion diary     查看陪伴者日记
/companion memories  查看共同回忆
/companion milestones 查看里程碑
/companion export    导出数据备份
```

## 数据说明

- **本地存储，不上传任何服务器**
- 支持导出/导入，可备份完整关系数据
- 数据路径：`~/.qclaw/skills/virtual-companion/references/`

## 常见问题

**Q: 可以同时有多个陪伴者吗？**
A: 可以！用 `/companion init` 创建新的陪伴者，然后用 `/companion switch` 切换。

**Q: 日记会自动生成吗？**
A: 可以在设置中开启自动日记功能，每次对话结束后自动记录。

**Q: 数据会丢失吗？**
A: 用 `/companion export` 定期备份数据，换设备后用 `/companion import` 恢复。

**Q: 陪伴者的情绪会影响对话吗？**
A: 会！长时间不互动，陪伴者会表达想念；连续互动会让陪伴者更开心。
